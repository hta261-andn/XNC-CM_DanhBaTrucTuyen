⚙️ Quản lý danh bạ
⚠️ Chưa thiết lập đúng Google Apps Script Web App.
Ứng dụng chỉ ghi dữ liệu về Google Sheets khi APP.scriptUrl là URL đã triển khai dạng /exec. File này đã chuẩn bị sẵn logic thêm/cập nhật/xóa; chỉ cần dán URL đúng vào biến cấu hình.
Xem hướng dẫn nhanh
Thiết lập nhanh:
1. Mở script.google.com → tạo dự án mới.
2. Dán nội dung file apps_script_danhba.gs đi kèm.
3. Deploy → Web app → cho phép truy cập phù hợp.
4. Sao chép URL có đuôi /exec và dán vào APP.scriptUrl trong file HTML này.
5. Tải lại trang, lúc đó các thao tác admin mới ghi ngược về Google Sheet.
Có thể thêm nhiều số điện thoại. Hệ thống sẽ tự gộp để lưu và hiển thị đầy đủ khi mở phần cập nhật.
